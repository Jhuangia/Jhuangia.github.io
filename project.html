Projects

Tell-A-Vision

(HackUCI 2022 Best Overall Hack [Runner Up])

An application that transcribes audio stories and creates a moving storyboard on the fly. Each storyboard is equipped with images,voices, and a sound track that fits the context of the story.

Built in a Vuejs frontend, and a backend server that uses Express and Socket.IO

The application used the Web Speech API, Google’s Natural Language API, Google’s Text to Speech API, and the Contextual Web Search API

pomofomo

(HackUCI 2021 Winner for Best Use of Twilio)

A web application for studying socially distanced with friends during the pandemic.
